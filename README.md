# Il2cpp-Trace-Module
一个用于对unity il2cpp框架开发的安卓端手游进行trace的so模块


# 如何使用？
使用方法目前和[frida-il2cpp-trace-module](https://github.com/AndroidReverser-Test/frida-il2cpp-trace-module)保持一致


# 已知的问题
部分游戏可能会在trace时崩溃，这与hook框架有关
